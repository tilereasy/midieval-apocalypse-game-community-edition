Игра Midieval Apocalypse является непревзойденной классикой и до сих будоражит умы геймеров по всему миру.
Поэтому обширным сообществом этой игры было принято решение сделать ПЕРЕИЗДАННУЮ ВЕРСИЮ для всех современных
игровых платформ(кроме консолей). Лучшие геймдизайнеры изменят многие механики этой замечательной игры и исправят баланс.
Кроме того Midieval Apocalypse Community Edition являетс ОПЕНС-СУРС проектом, так что ждем вашего вклада в разработку игры! 
На данный момент присоединится к разработке пообещали Илон Маск, Майкл Бэй, Линус Торвальдс, Билл Гейтс, Гейб Ньюэлл, ТОДД ГОВАРД. Все заработанные средства пойдут МНЕ.

Midieval Apocalypse is an unrivaled classic and still captivates gamers worldwide.
That's why the game's vast community has decided to create a RE-RELEASED VERSION for all modern gaming platforms (except consoles). Top game designers will tweak many mechanics of this wonderful game and improve its balance.
Furthermore, Midieval Apocalypse Community Edition is an OPEN-SOURCE project, so we welcome your contributions to the game's development!
Currently, Elon Musk, Michael Bay, Linus Torvalds, Bill Gates, Gabe Newell, and TODD HOWARD have pledged to join the development. All proceeds will go to me.
